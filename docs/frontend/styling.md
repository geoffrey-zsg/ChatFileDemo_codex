# styling.md

## 样式策略
- 使用 CSS variables 定义颜色、半径、阴影和布局尺寸。
- Tailwind 类负责组合，不把复杂视觉参数散落到业务 JSX。
- 动画统一沉淀在 `globals.css` 或共享样式模块。

## 主题基线
- 背景：暖白和浅灰分层。
- 强调色：青蓝色系。
- 高亮色：黄、蓝、绿三组引用色循环。
- 文本：深石墨主文本，灰色辅助文本。

## 响应式
- PC 保持双栏。
- 平板和手机下预览区切换为抽屉或覆盖层。
- 输入区和顶部栏在窄屏下仍保持主要操作可达。
